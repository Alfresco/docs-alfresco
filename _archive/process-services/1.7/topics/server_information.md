# Server Information

To retrieve information about the Process Services version, use the following command:

```
GET api/enterprise/app-version
```

**Response:**

```
{
   "edition": "Alfresco Activiti Enterprise BPM Suite",
   "majorVersion": "1",
   "revisionVersion": "0",
   "minorVersion": "2",
   "type": "bpmSuite",
}
```
