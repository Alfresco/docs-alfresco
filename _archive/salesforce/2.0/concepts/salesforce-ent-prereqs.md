# Prerequisites for Alfresco Content Connector for Salesforce

There are a number of software requirements for using Alfresco Content Connector for Salesforce.

**Alfresco requirements**

Alfresco One 5.0.3 or later

**Note:** Metadata mapping is available with Alfresco One 5.1 and later versions only.

**Salesforce requirements**

Salesforce Group is the minimum requirement. See [Salesforce Connected Apps](https://help.salesforce.com/apex/HTViewHelpDoc?id=connected_app_overview.htm) for guidance on levels of Salesforce required with connected apps.

**Java requirements**

Java 8 is recommended.

**Note:** If you are running an earlier version of Java, you might have problems mapping metadata and objects in the Alfresco Setup tab in Salesforce. See [TLS requirements](salesforce-troubleshoot.md#5) in the troubleshooting topic for more information.

**Other requirements**

Your Salesforce and Alfresco instances must be on a shared network or otherwise accessible in order to share information.
