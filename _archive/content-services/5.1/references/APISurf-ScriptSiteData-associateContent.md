# `associateContent`

The `associateContent` method associates content.

## Parameters

-   **contentId**

    A string representing the content id.

-   **templateId**

    A string representing the template id.

-   **assocType**

    A string representing the association type.

-   **formatId**

    A string representing the format id.


## Returns

void
