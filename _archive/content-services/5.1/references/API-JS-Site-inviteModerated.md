# `inviteModerated`

The `inviteModerated(inviteeComments, inviteeUserName, inviteeRole)` creates a new moderated invitation to the specified web site.

## Parameters

-   **inviteeComments**

    String.

-   **inviteeUserName**

    String.

-   **inviteeRole**

    String.


## Returns

A `ScriptInvitation` object.
