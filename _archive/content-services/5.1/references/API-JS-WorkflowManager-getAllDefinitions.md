# `getAllDefinitions`

`getAllDefinitions()` - Returns all versions of the deployed workflow definitions.

## Parameters

None

## Returns

Returns all versions of the deployed workflow definitions.

## Example

```
    
model.definitions = workflow.getAllDefinitions();        
      
```
