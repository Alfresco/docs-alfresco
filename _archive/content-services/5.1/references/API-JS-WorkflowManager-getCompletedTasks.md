# `getCompletedTasks`

`getCompletedTasks()` - Get completed tasks assigned to the current user.

## Parameters

None

## Returns

Returns the list of completed tasks.

## Example

```

    model.completedTasks = workflow.getCompletedTasks();          
      
```
