# Deleting a document

Delete a document to permanently remove it from the repository.

1.  On the Browse Spaces and Documents tab of the Alfresco panel, navigate to the document you want to delete.

    **Note:** You cannot delete the working copy of a document.

2.  Select ![Delete](../images/MSAddin-delete.png) \(**Delete**\) associated with the document you want to remove.

3.  Click **OK** when prompted to confirm the deletion.

    The document is deleted from the repository and removed from the document list.
