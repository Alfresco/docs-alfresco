# Transforming a document to PDF

You can easily transform a Word document in the Alfresco repository into Adobe PDF format.

1.  On the Browse Spaces and Documents tab of the Alfresco panel, navigate to the document you want to transform.

    You can also perform this action on the My Alfresco and Document Details tabs.

2.  Click ![Transform to PDF](../images/MSAddin-PDFtransform.png) \(**Transform to PDF**\) associated with the document you want to transform.

    The PDF version of the document is created and added to the same space as the original document.
