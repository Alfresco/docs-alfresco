# Viewing document tags

Viewing tags enables you to filter the repository contents to make locating specific documents more manageable.

1.  Click the Document Tags tab on the Alfresco panel to view all of the tags associated with the documents in Alfresco.

2.  In the **Tag Cloud** pane, click a tag to display all documents and files associated with the tag.

3.  In the list returned, click any displayed document to open it.
