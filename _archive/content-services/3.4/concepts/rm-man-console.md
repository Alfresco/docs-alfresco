# Management Console

The Management Console allows you to manage the Records Management site.

You can manage the following Records Management-specific administration tasks:

-   Audit
-   Custom Metadata
-   Define Roles
-   Email Mappings
-   Events
-   List of Values
-   Relationships
-   User Rights Report
