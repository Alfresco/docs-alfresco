# PickerList

Service to return metadata of child items in a parent space to display in the object finder.

`GET /alfresco/service/api/forms/picker/{type}/{store_type}/{store_id}/{id}/children`

`GET /alfresco/service/api/forms/picker/{type}/{store_type}/{store_id}/{id}/siblings`

`GET /alfresco/service/api/forms/picker/{type}/{store_type}/{store_id}/{id}/doclib`

`GET /alfresco/service/api/forms/picker/{type}/children`

The web script description document specifies the following options:

|`json`|The default response format|
|`user`|The authentication access|
|`required`|The transaction level|
|`any`|The format style|
