# Get rules collection

Gets a list of all owned rules for a given node.

`GET /alfresco/service/api/node/{store_type}/{store_id}/{id}/ruleset/rules?ruleType={rule_type?}`

The node can be specified either using a node reference or a query path.

The web script description document specifies the following options:

|`json`|The default response format|
|`user`|The authentication access|
|`required`|The transaction level|
|`argument`|The format style|
