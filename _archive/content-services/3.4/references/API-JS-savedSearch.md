# ``savedSearch``

`savedSearch(node)`

This method returns an array of `ScriptNode` objects that were found by executing the Saved Search referenced by the supplied `node` object.

## Parameters

-   **node**

    The node object


## Returns

Array of `ScriptNode` objects

`savedSearch(noderef)`

    The noderef string
