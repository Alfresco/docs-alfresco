# `setIsTagScope`

`setIsTagScope(boolean value)`Sets whether this node is a tag scope or not.

## Parameters

-   **value**

    True if this node is a [TagScope](API-JS-TaggingService-tagScope.md), false otherwise.


## Returns

void
