# Node Locator

Locates a Node in the repository using the specified Node Location strategy.

`GET /alfresco/service/api/{store_type}/{store_id}/{node_id}/nodelocator/{node_locator_name}`

`GET /alfresco/service/api/nodelocator/{node_locator_name}`



The web script description document specifies the following options:

|`json`|The default response format|
|`user`|The authentication access|
|`required`|The transaction level|
|`any`|The format style|
