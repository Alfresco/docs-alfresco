# Update a persisted replication definition

Updates a previously persisted replication defintion with the supplied details.

`PUT /alfresco/service/api/replication-definition/{replication_definition_name}`

The web script description document specifies the following options:

|`json`|The default response format|
|`admin`|The authentication access|
|`required`|The transaction level|
|`any`|The format style|
