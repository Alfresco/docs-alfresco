# `inviteModerated`

`inviteModerated(inviteeComments,inviteeUserName,inviteeRole)` this method creates a new moderated invitation to this web site.

## Parameters

-   **inviteeComments**

    String.

-   **inviteeUserName**

    String.

-   **inviteeRole**

    String


## Returns

A `ScriptInvitation` object.
