# Delete rule

Deletes a rule.

`DELETE /alfresco/service/api/node/{store_type}/{store_id}/{id}/ruleset/rules/{rule_id}`

|`json`|The default response format|
|`user`|The authentication access|
|`required`|The transaction level|
