# Welcome

Welcome to Alfresco documentation.

**Tip:** Looking for the online documentation for a previous Alfresco release? Go to [Alfresco documentation online](../../) to find all the documentation resources.

The documentation for older service packs can also be found on the [Alfresco Support Portal](http://support.alfresco.com).

Alfresco customers can also go to the [Alfresco Support Handbook](http://docs.alfresco.com/support/concepts/su-welcome.html).

For help with Alfresco Community, go to the [Alfresco Community Documentation](../../community/concepts/welcome-infocenter_community.html) site.

Alfresco is an open source Enterprise Content Management \(ECM\) system that manages all the content within an enterprise and provides the services and controls that manage this content.
