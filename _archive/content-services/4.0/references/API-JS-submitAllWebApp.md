# `submitAllWebApp`

`submitAllWebApp(webApp,submitLabel,submitComment)` submits the modified items for the specified `webApp` directly to staging bypassing any workflows that may have been configured for the web project.

## Parameters

-   **webApp**
-   **submitLabel**
-   **submitComment**
