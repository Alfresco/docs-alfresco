# Working with the Search Service

The Search Service page in Admin Console enables you to manage and monitor the search service you want to use in Alfresco.

You can choose any one of the following search service for use in Alfresco.

-   Solr search service
-   Lucene search service
-   No index

The Admin Console enables you to configure the search services using configuration properties. Click on the relevant search service for more information.

-   **[Configuring the Solr search service](../tasks/adminconsole-searchservice-solr.md)**  
The topic describes the properties for configuring the Solr search service.
-   **[Configuring the Lucene search service](../tasks/adminconsole-searchservice-lucene.md)**  
The topic describes the properties for configuring the Lucene search service.
-   **[Configuring No Index search service](../tasks/adminconsole-searchservice-noindex.md)**  
The topic describes the properties for configuring the No Index search service.
