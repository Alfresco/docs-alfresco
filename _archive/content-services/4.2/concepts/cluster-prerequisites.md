# Cluster configuration prerequisites

This topic describes the prerequisites for setting up an Alfresco, Version 4.2, repository server cluster.

The repository server cluster consists of the following components:

-   Database server
-   Content store, for example, NFS server
-   Load balancer
-   Hazelcast mancenter server \(optional\)
