# Channel Authorisation Form

Used to obtain a form to provide credentials to publish to a channel.

`GET /alfresco/service/api/publishing/channels/{store_protocol}/{store_id}/{node_id}/authform`



The web script description document specifies the following options:

|`html`|The default response format|
|`user`|The authentication access|
|`required`|The transaction level|
|`any`|The format style|
