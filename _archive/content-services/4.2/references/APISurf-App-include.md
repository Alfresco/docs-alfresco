# `include`

`include(String relativePath)` - performs a server-side include of a web asset.

## Parameters

-   **relativePath**

    A string representing the relative path.


## Returns

The result string is returned. Valid paths are, for example:

```

/a/b/c.gif
/images/test.jpg

```
