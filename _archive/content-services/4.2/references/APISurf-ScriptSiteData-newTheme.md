# `newTheme`

`newTheme(String objectId)` - this method returns a newly created `ScriptModelObject` representing a new Theme.

## Parameters

-   **objectId**

    A string representing the object ID.


## Returns

Returns a `ScriptModelObject` representing a new Theme.
