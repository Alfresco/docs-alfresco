# Get Publishing Channels

Get the publishing channels for specified NodeRef or returns all Publishing Channels.

`GET /alfresco/service/api/publishing/channels`

`GET /alfresco/service/api/publishing/{store_protocol}/{store_id}/{node_id}/channels`

The web script description document specifies the following options:

|`json`|The default response format|
|`user`|The authentication access|
|`required`|The transaction level|
|`any`|The format style|
