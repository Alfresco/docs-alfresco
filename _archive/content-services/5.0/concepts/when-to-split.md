# When to split

This topic provides indicators to help you decide when to split your architecture from a single node environment to a distributed node environment.

Some of the indicators to look for include:

-   Low disk space
-   CPU running out of memory
-   High indexing load

**Parent topic:**[Recommendations for split architecture](../concepts/recommend-split.md)
