# `unassociateContentType`

`unassociateContentType` - unassociates content type.

## Parameters

-   **contentTypeId**

    A string representing the content type id.

-   **templateId**

    A string representing the template id.

-   **formatId**

    A string representing the format id.


## Returns

void
