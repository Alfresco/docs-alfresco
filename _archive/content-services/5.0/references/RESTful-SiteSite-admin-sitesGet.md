# Get sites

Get a collection of the sites in the repository accessible to site administrators.

`GET /alfresco/service/api/admin-sites?nf={namefilter?}&maxItems={maxItems?}&skipCount={skipCount?}&roles={roles?}&`

The collection can be filtered by name.

The web script description document specifies the following options:

|`json`|The default response format|
|`user`|The authentication access|
|`required`|The transaction level|
|`argument`|The format style|
