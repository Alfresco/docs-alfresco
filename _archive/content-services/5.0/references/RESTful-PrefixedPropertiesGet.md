# Get property definitions

Gets the collection of property definitions.

`GET /alfresco/service/api/defclasses/{prefix}/{shortClassName}/properties?nsp={namespacePrefix?}&n={name?}`

`GET /alfresco/service/api/defproperties?nsp={namespacePrefix?}&n={name?}`

The web script description document specifies the following options:

|`json`|The default response format|
|`user`|The authentication access|
|`required`|The transaction level|
|`argument`|The format style|
